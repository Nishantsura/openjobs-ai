# OpenJobs AI Privacy Policy

## Data Collected
- Profile details entered in onboarding (name, email, phone, location, preferences).
- Resume PDF uploaded by the user.
- Generated content requests (email, cover letter, form answers).
- Operational logs (endpoint, status, latency, error code, trace id).

## Purpose of Data Use
- Autofill job application forms.
- Generate user-requested AI outputs.
- Improve reliability and debug failures.

## Storage and Retention
- Extension local storage: onboarding profile, feature flags, and resume file cache.
- Backend storage (Supabase): profile JSON, resume files, generation logs.
- Retention policy: operational logs retained for support and reliability analysis; users can request deletion.

## Third-Party Processors
- Supabase (auth, database, storage).
- OpenRouter (AI inference routing and model access).

## User Rights and Deletion
- Users can request account/profile deletion and resume removal via support channel.
- Deletion requests remove profile data and stored resume artifacts associated with the user id.
